# Lazarus Group Cyber Threat Intelligence Dossier

This project is an independent cyber threat intelligence dossier on Lazarus Group, a DPRK-linked advanced persistent threat actor. It analyzes attribution, MITRE ATT&CK TTPs, malware, campaigns, IOC enrichment, infrastructure patterns, and defensive recommendations using public OSINT sources.

## Project Overview

The objective of this project was to produce a portfolio-grade threat intelligence report that profiles Lazarus Group from both a strategic and tactical intelligence perspective. The report covers actor attribution, motivation, target scope, campaign evolution, malware ecosystem, infrastructure behavior, and defensive priorities.

## Key Areas Covered

* DPRK attribution and state nexus
* MITRE ATT&CK technique mapping
* Campaign timeline from 2009–2026
* Malware and tooling analysis
* Infrastructure and command-and-control patterns
* IOC enrichment using VirusTotal
* Target geography and sector analysis
* Defensive recommendations for SOC and security teams

## Methodology

This project uses publicly available intelligence from MITRE ATT&CK, CISA, FBI, DOJ, U.S. Treasury, Microsoft Security, and OSINT enrichment tools. Selected indicators were manually reviewed through VirusTotal to support tactical IOC analysis.

## Disclaimer

This project was created for educational and portfolio purposes. It uses only public, open-source information and does not include classified, private, or proprietary intelligence.
